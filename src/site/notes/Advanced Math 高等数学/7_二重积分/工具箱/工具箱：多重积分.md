# 二重积分
### 对称性

观察积分，判断是否存在对称性并计算
可将函数的乘积乘起来，使用对称性

### 选择坐标系
根据题目和区域，选择适当的坐标系
常见的：
直角坐标
极坐标
换元法

注意：转换坐标系后要乘上雅可比行列式
### 划分区域
注意是否分段讨论

### 对称性
再次观察坐标变换后的对称性
尽可能利用对称性，将积分限制到第一卦限，避免出错

# 三重积分

### 对称性
观察积分，判断是否存在对称性并计算
可将函数的乘积乘起来，使用对称性
### 选择积分顺序
分为
1. 外一内二：$\iiint_{\Omega} f(x,y,z)dV = \int ^{b}_{a}dz \iint_{D} f(x,y,z)d\sigma$
   使用时机：
	1. 被积函数只有一个自变量
	2. 其中一个自变量的范围是被常数所固定的
2. 外二内一：$\iiint_{\Omega} f(x,y,z)dV =  \iint_{D} d\sigma \int ^{z_{2}(x,y)}_{z_{1}(x,y)}f(x,y,z)$
   使用时机：
3. 坐标变换
4. 特殊换元

### 划分区域

注意是否分段讨论

### 对称性
再次观察坐标变换后的对称性
尽可能利用对称性，将积分限制到第一卦限，避免出错
